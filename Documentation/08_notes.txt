ANALYSIS 2 – Snapshot Date Creation
Objective :
Create a reference date for calculating customer recency.

Method:
The latest transaction date in the dataset was identified and one day was added to create a snapshot date.

Purpose:
The snapshot date acts as the reference point for measuring how recently customers made purchases.

Interpretation:
Customers with smaller recency values are more active than customers with larger recency values.

Conclusion:
The snapshot date was successfully established for recency calculations.

ANALYSIS 3 – Recency Calculation
Objective:

Method:
